Fedora: My First Distro Hop

Not even a week later. Sorry Pop!_OS, I was too curious.
20 Feb, 2024 at 13:48 · 2 min · 214 words · Ewan

Yep.

I decided to make the jump since I wanted to branch out from solely Ubuntu-based distributions, and Fedora (Workstation) seemed like a good choice.

It’s a part of the Red Hat lineage, so it has a different base from Ubuntu and its derivatives, which are based on Debian instead. I have looked into it around the internet and it seems to be one of the most stable distros.

As far as I can tell, right now, the only real difference is the use of the DNF package manager rather than APT. This doesn’t change much, and I am still able to use the homebrew package manager alongside it.
