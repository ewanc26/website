# Ewan's Website

Personal website powered by AT Protocol, built with SvelteKit 2 and Tailwind CSS 4.

**Version**: 11.7.2

Live at [ewancroft.uk](https://ewancroft.uk).

## Documentation

Full documentation is available at [docs.ewancroft.uk](https://docs.ewancroft.uk).

For local setup, see the [Configuration Guide](./docs/configuration.md).

## Quick Start

```bash
git clone git@github.com:ewanc26/website.git
cd website
npm install
npm run dev
```

## Features

- AT Protocol integration (Bluesky profile, posts, music status)
- Standard.site publishing support
- 12 color themes with dark mode
- Responsive design with SEO optimization
